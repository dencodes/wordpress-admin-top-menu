Move any menu items from the sidebar to the admin bar - into the new dropdown Top-Menu.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/admin-top-menu` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `Settings / Admin Top-Menu` screen to setup the Top-Menu.

== Screenshots ==

1. Select the sidebar items.
2. Now these elements are located in the top drop-down menu.
