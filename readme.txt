Move any items from the admin sidebar-menu to the new dropdown Top-Menu, located in the Admin Bar.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/admin-top-menu` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the `Settings / Admin Top-Menu` screen to select menu-items for Top-menu.
